# Ejecutar en desarrollo

1. Clonar el repositorio.

2. Ejecutar el siguiente comando:
```
npm install
```

3. Garantizar que tenga instalado Nest CLI.
```
npm install -g @nestjs/cli
```

4. Levantar la base de datos.
```
docker-compose up -d
```

## Stack usado.

* MongoDB.
* Nestjs.
